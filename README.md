## 🚀 Featured Project

<div align="center">

<a href="https://github.com/nps-genomics/GenoBridge">
  <img src="https://img.shields.io/static/v1?label=Featured%20Project&message=GenoBridge&color=0f172a&labelColor=3fb950&style=for-the-badge" alt="GenoBridge badge" />
</a>

</div>

**GenoBridge** is a sample-size-adaptive ensemble ML pipeline for genotype-to-phenotype prediction using SNP data from VCF files. It addresses three core limitations in current genomic prediction workflows:

- 🔷 **Adaptive model selection** — automatically selects the optimal ML model class (Ridge, Random Forest, XGBoost, MLP, Stacking Ensemble) based on cohort size
- 🔷 **CV-R² diagnostics** — distinguishes true predictive generalization from rank-order preservation
- 🔷 **ML-gated GWAS** — concentrates statistical power on traits with evidence of genomic predictability

> 🧬 Supports **30–100,000+ samples** across any plant species · No Python required · Standalone binaries for Linux, macOS, CentOS

### 🍊 Citrus Genomics
Complete chloroplast genome characterization of three red-fleshed grapefruit cultivars (*Rio Red, Ruby Red, Texas Red TR-1*) via **PacBio HiFi WGS**. Resolving IR boundary polymorphisms and investigating plastid-to-mitochondria gene transfer relevant to **HLB disease** research.

</td>
<td width="50%">

### 🌾 Wheat Pangenomics
Assembling and analyzing pangenomes of Montana winter, spring, and durum wheat varieties using **whole-genome resequencing**. Characterizing selective sweeps, genomic diversity, and functional gene expansion across elite North American germplasm.

</td>
</tr>

### 🔗 Cross-Kingdom sRNA Regulation
Investigating **microRNA-mediated cross-kingdom gene silencing** between wheat and wheat stem sawfly (*Cephus cinctus*). Host-derived sRNAs as a novel mechanism of crop resistance.
